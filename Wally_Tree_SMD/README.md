# Wally Tree Driver

The "Wally Tree Driver" is a Differental Reciver board designed to control at 5" Holiday Time RGB Tree from Walmart[Link](https://www.walmart.com/ip/Holiday-Time-5-Rgb-Color-Changing-Pixel-Tree/908536686). All the design files are open source and available on github.

To order PCBs upload the "GERBER-Wally_Tree.zip" file in the "jlcpcb\gerber" folder. Enable the SMD service, 'BOM-Wally_Tree.csv' is the SMD BOM and 'POS-Wally_Tree.csv' is the SMD CPL position file.
